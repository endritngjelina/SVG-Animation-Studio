# 🎨 SVG Animation Studio

A lightweight, browser-based tool for creating and previewing animated SVG graphics. Built entirely with vanilla HTML, CSS, and JavaScript — no frameworks, bundlers, or dependencies required.

---

## 🚀 Features

- Create and animate SVG shapes directly in the browser
- Real-time animation preview
- Customize animation timing, duration, and easing
- Iterative design explorations across 6 versions (v0–v5)
- Zero setup — open any file and start animating instantly

---

## 🛠️ Usage

1. Clone or download the repository:
   ```bash
   git clone https://github.com/endritngjelina/SVG-Animation-Studio.git
   ```
2. Open any `.html` file directly in your browser — no server needed.
3. Recommended: Open **SVG animation studio 5.html** for the most complete experience.

---

## 💡 Use Cases

- Prototyping SVG animations for web projects
- Learning CSS and SVG animation techniques
- Creating animated icons, loaders, and UI elements
- Experimenting with keyframes and SVG path morphing
